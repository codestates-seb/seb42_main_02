## 🤍 서비스 소개 

### ‘다나아’에서 건강 고민 ‘다 나았다!’
>‘다나아’는 일반인과 의료인이 함께 소통하는 의료 커뮤니티 플랫폼 입니다.                                                              
>‘다나아’는 가족 혹은 친구에게도 말 못하는 건강 고민을 커뮤니티에 남기고 의료 전문가 및  여러 사람들과 소통할 수 있습니다.                                                
>‘다나아’는 소비자들이 직접 가본 솔직한 후기를 공유할 수 있습니다.
